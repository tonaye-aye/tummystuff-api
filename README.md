# Backend

This is the backend of a full stack blog site built with Next.js, Graph QL, Strapi amd MongoDB, designed with Material UI and deployed using Heroku, Amazon S3 and Vercel.

The backend is delivered using;

- Strapi
- MongoDB
- Heroku
- Amazon S3
